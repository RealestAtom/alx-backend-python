# Python Generators - Database Seeder

## Description
This script (`seed.py`) sets up a MySQL database named **ALX_prodev**, creates a table `user_data`, loads records from a CSV file, and provides a generator to stream table rows one at a time.

## Files
- `seed.py` — Main script for database setup and data seeding
- `user_data.csv` — Input data file
- `0-main.py` — Test file provided by ALX

## Usage
1. Start MySQL server
2. Update your MySQL credentials in `seed.py`
3. Run the test script:
   ```bash
   ./0-main.py
